AndroidUserInfo

用户安卓信息表
只在首次微信授权登录或绑定手机号时获取。

考虑用户可能在不同机器上登录，于是将用户唯一标识和安卓ID同时设置为主键。
当检测到AndroidID更换时，再次插入数据

字段名称 主键 类型 长度 允许为空 默认值 描述 数值精度 标识列
ID 1 uniqueidentifier 16 用户唯一标识 0
AndroidID 1 varchar 50 用户Openid 0
CreateTime datetime 8 (getdate()) 用户昵称 3

Users

用户表
微信授权登录也会在这个表插入数据（此时只写入用户唯一标识信息），当已授权用户绑定手机号再重新写入其它信息。

字段名称 主键 类型 长度 允许为空 默认值 描述 数值精度 标识列
ID 1 uniqueidentifier 16 用户唯一标识 0
Phone varchar 50 1 手机号码 0
Password varchar 50 1 登录密码 0
Salt char 6 1 盐值 0
SaltPassword char 40 1 加密后密码 0
RegisterTime datetime 8 1 首次注册时间 3
CreateTime datetime 8 (getdate()) 首次授权或注册时间，插入数据库当时的时间 3

WXUserInfo

用户微信信息表
在用户同意微信授权登录时获取。

字段名称 主键 类型 长度 允许为空 默认值 描述 数值精度 标识列
ID 1 uniqueidentifier 16 用户唯一标识 0
Openid varchar 50 1 用户Openid 0
NickName nvarchar 100 1 用户昵称 0
Sex tinyint 1 1 用户性别，值为1时是男性，为2时是女性，为0时是未知 0
City nvarchar 100 1 用户所在城市 0
Country nvarchar 100 1 用户所在国家 0
Province nvarchar 100 1 用户所在省份 0
Language nvarchar 100 1 用户的语言 0
HeadImgUrl nvarchar 1000 1 用户头像图片链接 0
